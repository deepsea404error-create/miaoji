# Miaoji（喵吉）— 饥荒联机版人物 Mod

> 作者：**DeepSea404**  
> 版本：**0.0.6**  
> 基于 [喵吉（单联机通用版）](https://steamcommunity.com/sharedfiles/filedetails/?id=1296609863) 的全面重造

---

## 📖 简介

*"这里是哪里喵"*

一只普通的猫娘穿越到了永恒领域，在黄昏与黑夜中逐步觉醒，获得力量。

---

## ✨ 角色特性

| 特性 | 说明 |
|------|------|
| **等级成长** | 通过击杀和进食获得经验，提升三围上限、伤害和移速 |
| **黑暗亲和** | 黄昏/夜晚/洞穴中获得伤害加成和理智恢复 |
| **夜视能力** | 夜晚和洞穴中自带可成长光源 |
| **免疫黑暗** | 完全免疫黑暗理智流失 |
| **免疫腐烂** | 腐烂食物无任何负面效果 |
| **鱼类专精** | 鱼类及海鲜料理效果大幅增强，并提供经验 |
| **免疫诅咒** | 完全抵抗猴子诅咒 |
| **书籍系统** | 达到等级后可制作和阅读薇克巴顿的书籍 |

---

| 方式 | 经验 |
